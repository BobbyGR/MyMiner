Here you will have the ability to add a pool/coin to your Rig/s with a few simple clicks.  Before you get started you need to be aware of how this process works. Incorrect settings could result in a weird mining/not mining state.

  • You can only select the all rigs button if each rig is online, and not processing anothe request.  this is done in batches and to make sure it's done properly they all need to be in the same state. Rigs must be in an OK state to apply the settings.
  • All pool settings must be tested before they can be added, we will perform a simple stratum test to ensure your pool is operational.  If you put in incorrect information and start mining, you could put your mining rig at risk of a failure or even worse...downtime
  • Any pool that you do select for a coin here it will become the primary pool for that coin.  If you need to change it back, you would either need to re-add it or change it inside of AIOMiner
